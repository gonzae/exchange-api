# exchange-api

Just a small exchange rate api.

## Before starting
Make sure you have already installed:
* [npm](https://www.npmjs.com/) >= v6.14.5
* [NodeJS](https://nodejs.org/) >= v12.17.0

## Installation
Clone this repo.

Then, install dependencies:
```
npm i
```

## Run the app
Before running the app, make sure you have a [Fixer](https://fixer.io/) access key. Add it as environment variable as `FIXER_ACCESS_KEY`. Or you can opt for running the app like this:
```
FIXER_ACCESS_KEY='<yourFixerIoAccessKey>' npm start
```

Enjoy at http://localhost:3001/
